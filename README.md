# VISIORYX PRO | Neural Forensic Suite 🛡️

Visioryx is a high-performance Multimodal Intelligence engine designed to bridge the gap between **Neural Scene Recognition** and **Digital Forensics**. It analyzes visual assets to provide descriptive intelligence while performing a deep scan for hardware metadata.

## 🛠️ Key Features
* **Neural Scene Captioning:** Utilizes the Salesforce BLIP model for high-accuracy descriptive analysis.
* **Forensic Metadata Extraction:** Deep-scans EXIF headers for Camera Make, Model, ISO, and Aperture.
* **Privacy Scrub Detection:** Deliberate logic to identify when metadata has been stripped by source platforms (e.g., Unsplash, Facebook).
* **Automated Reporting:** Generates a professional forensic PDF report for every analyzed asset.

## 💻 Technical Stack
* **Language:** Python 3.9+
* **AI Models:** Transformers (BLIP-Base)
* **Backend:** FastAPI / Uvicorn
* **Frontend:** Tailwind CSS / Glassmorphism UI
* **Libraries:** PyTorch, Pillow, FPDF2
